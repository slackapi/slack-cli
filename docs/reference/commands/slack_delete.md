# `slack delete`

Delete the app

## Synopsis

Uninstall the app from the team and permanently delete the app and all of its data

```
slack delete [flags]
```

## Examples

```
# Delete an app and app info from a team
$ slack app delete

# Delete a specific app from a team
$ slack app delete --team T0123456 --app local
```
